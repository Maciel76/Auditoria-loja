# APRESENTAÇÃO DE VENDA — Sistema de Auditoria Inteligente para Lojas

## PARTE 1: O PROBLEMA (Comece por aqui no vídeo)

---

### O cenário real que toda rede de lojas vive hoje:

Imagine o seguinte: você é gestor de uma grande rede de Atacadista e essas lojas precisam realizar auditorias todos os dias da semana para verificar se os produtos estão corretamente precificados e disponíveis nas gôndolas. e cada uma dessas lojas possui cerca de 9 mil itens que precisam ser verificados diariamente. e para verifica todos esses itens a loja precisa disponibilizar metades dos funcionários da loja Todo os  dias,  ate ai tudo bem os colaboradores fazem auditorias — verificam etiquetas de preço, checam se os produtos estão presentes nas gôndolas, identificam rupturas de estoque. Tudo isso gera **planilhas enormes**. Centenas, milhares de linhas. mas o que 

E o que acontece com esses dados?

- **Ficam perdidos em pastas do computador**
- **Ninguém cruza as informações entre lojas**
- **Não existe visão em tempo real do desempenho**
- **Não há como saber qual colaborador está performando melhor**
- **Não há como comparar lojas entre si**
- **O gestor só descobre problemas quando já é tarde demais**
- **Zero motivação para o colaborador fazer auditoria com qualidade**

### As dores reais:

| Problema | Impacto direto |
|----------|---------------|
| Planilhas espalhadas sem centralização | Perda de informação, retrabalho, decisões no escuro |
| Sem ranking de desempenho | Colaboradores não sabem onde melhorar, gestores não sabem quem premiar |
| Sem análise por setor/corredor | Não identifica áreas críticas da loja |
| Sem comparativo entre lojas | Impossível saber qual loja precisa de atenção |
| Sem gamificação | Colaboradores fazem auditoria por obrigação, sem engajamento |
| Processo 100% manual | Tempo gasto em análise que poderia ser automatizada |

### A pergunta que o gestor deveria fazer:

> "Quanto dinheiro minha loja perde por NÃO ter visibilidade em tempo real das auditorias? Quantos produtos com preço errado estão na gôndola agora? Quantas rupturas estão acontecendo sem que eu saiba?"

## PARTE 2: A SOLUÇÃO — O que é o Sistema

---

### Fala sugerida para o vídeo:

> "Eu criei uma plataforma completa que transforma dados brutos de auditoria em inteligência visual, rankings, métricas e gamificação. Tudo automatizado. Você só precisa fazer o upload da planilha."

### O que é o sistema:

Um **sistema web completo de gestão e análise de auditorias para lojas**, que permite:

1. **Centralizar** todos os dados de auditoria em um único lugar
2. **Visualizar** métricas e estatísticas instantaneamente
3. **Comparar** desempenho entre colaboradores e entre lojas
4. **Engajar** equipes com sistema de conquistas, XP e níveis
5. **Decidir** com base em dados reais, não em achismo

### Stack Tecnológica (mencione brevemente):

- **Frontend:** Vue.js 3 com Vite (rápido, moderno, responsivo)
- **Backend:** Node.js com Express (robusto, escalável)
- **Banco de Dados:** MongoDB Atlas (nuvem, seguro, escalável)
- **Deploy:** Vercel (frontend) + Render (backend) — pronto para produção

### CENA 9.1 — Como Funciona o Cálculo do Ranking

> "Agora vou explicar como o sistema calcula o ranking, porque isso é importante você entender."
>
> "Para os tipos de auditoria **Presença** e **Etiqueta**, o ranking é baseado na **quantidade total de itens auditados**. Simples: quem audita mais itens, fica melhor posicionado."
>
> "Mas para **Ruptura**, é diferente. Além da quantidade de itens, o sistema calcula o **valor financeiro da ruptura que cada colaborador identificou e evitou**. Ou seja: se um colaborador encontra 100 produtos em ruptura que totalizavam R$ 5.000 em perdas, o sistema mostra esse valor no ranking. Isso deixa claro o impacto financeiro real do trabalho dele."
>
> "Agora vem a parte inteligente: **e se você enviar mais de uma planilha no mesmo dia?** O sistema tem um algoritmo que evita inflar os números. Quando você envia uma nova planilha no mesmo dia, o sistema **atualiza o ranking do dia, mas usa only a última planilha de cada dia**. Isso significa que se você enviar 3 planilhas em um mesmo dia, apenas a terceira conta para o ranking diário. Já para o **ranking do período completo**, o sistema também se atualiza automaticamente para refletir somente a última versão de cada dia."
>
> "Resultado? O ranking é sempre preciso, sempre reflete os dados mais recentes, e ninguém consegue inflar sua posição enviando múltiplas planilhas. Transparência e justiça total."
