# Covers para Perfil de Loja

Esta pasta contém as imagens de fundo (covers) disponíveis para os perfis das lojas.

## Estrutura

Cada cover deve ter:

- Dimensões recomendadas: 1920x400px
- Formato: JPG ou PNG
- Nome do arquivo: cover-1.jpg, cover-2.jpg, etc.

## Covers Disponíveis

1. **cover-1.jpg** - Gradiente Azul/Roxo (padrão)
2. **cover-2.jpg** - Gradiente Verde/Azul
3. **cover-3.jpg** - Gradiente Laranja/Rosa
4. **cover-4.jpg** - Gradiente Roxo/Pink
5. **cover-5.jpg** - Gradiente Vermelho/Laranja
6. **cover-6.jpg** - Padrão Geométrico Azul
7. **cover-7.jpg** - Padrão Ondas Verde
8. **cover-8.jpg** - Abstrato Roxo
9. **cover-9.jpg** - Minimalista Cinza
10. **cover-10.jpg** - Tecnológico Azul Escuro

## Como adicionar novos covers

1. Adicione a imagem nesta pasta
2. Nomeie seguindo o padrão: cover-[número].jpg
3. Atualize o array `covers` no componente `CoverSelector.vue`
